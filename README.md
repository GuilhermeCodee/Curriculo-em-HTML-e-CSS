📄 README — Portfólio Web (HTML + CSS)
📌 Descrição do Projeto

Este projeto consiste no desenvolvimento de um currículo/portfólio online, criado utilizando apenas HTML e CSS, com foco em apresentar informações pessoais, habilidades e experiências de forma moderna, organizada e responsiva.

O objetivo principal é demonstrar conhecimentos em estruturação de páginas web, estilização avançada e boas práticas de design, sem o uso de JavaScript.

🎯 Objetivos
Criar uma página web responsiva
Aplicar conceitos de HTML semântico
Utilizar CSS moderno (Flexbox, gradientes, animações)
Desenvolver um layout visualmente atrativo
Exibir habilidades em formato interativo (skills animadas)
🛠️ Tecnologias Utilizadas
HTML5 → Estrutura da página
CSS3 → Estilização e animações
Flexbox → Organização de layout
Gradientes CSS → Visual moderno
Keyframes (CSS Animation) → Animações nas skills
💻 Funcionalidades
✅ Perfil com imagem e descrição
✅ Seção “Sobre mim”
✅ Experiência profissional
✅ Skills com animação circular (sem JavaScript)
✅ Ícones de redes sociais
✅ Layout moderno com efeito glassmorphism
✅ Design responsivo
🎨 Destaques do Projeto
Uso de conic-gradient para criar gráficos circulares de habilidades
Animações feitas 100% em CSS
Interface com efeito glass (blur + transparência)
Design limpo e profissional
